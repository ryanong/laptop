# Laptop

Laptop is an idempotent script to set up a Mac OS X laptop for Rails development.

## Install

### Mac OS X

Read, then run the script:

```sh
bash <(curl -s https://raw.githubusercontent.com/ryanong/laptop/master/mac)
```
